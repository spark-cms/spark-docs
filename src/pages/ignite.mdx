# Ignite
import { Callout } from 'nextra/components'

<Callout>
  Ignite is still under active development as SPARK is still in beta. The API is subject to change and bugs are to be expected.
</Callout>

**Ignite** is a data fetching library that allows you to fetch data from your SPARK project with ease.

import { Steps } from 'nextra/components'

<Steps>
### Install

```sh npm2yarn
npm i @spark-cms/ignite
```

### Import the package

```js filename="ignite.js" copy
import { Ignite } from "@spark-cms/ignite";
```

### Create a new Ignite instance

```js filename="ignite.js" copy
const spark = new Ignite({
  name: "John Doe",
  project: "Private Blog",
  token: "SPARK-a216ee62-b106-447d-98ff-e99cd70d3340",
});
```

### Fetch data from the CMS

```js filename="ignite.js" copy
const data = await spark.fetchContent("blog", "posts");
```
</Steps>
